## What is Xcode Pilot MCP?

Xcode Pilot MCP is a **Model Context Protocol server** that gives AI assistants like Claude complete control over the iOS and macOS development toolchain. Instead of copy-pasting terminal commands, just describe what you want in plain English.

```bash
# Install and start using immediately
npx xcode-pilot-mcp
```
