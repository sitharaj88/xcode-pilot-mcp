List all physical iOS devices currently connected to your Mac via USB or paired over Wi-Fi. This tool reports each device's name, model, OS version, UDID, and connection state. Use it to discover device identifiers before installing, launching, or streaming console logs from a physical device.

## Parameters

This tool takes no parameters.

## Usage Examples

### List all connected devices

> "Show me all the physical devices connected to my Mac"

## Tips

<Aside type="tip">
  Run this tool first whenever you need to interact with a physical device. The UDID from the output is required by [`physical_device_install`](/xcode-pilot-mcp/tools/device/physical-device-install/), [`physical_device_launch`](/xcode-pilot-mcp/tools/device/physical-device-launch/), and [`physical_device_console`](/xcode-pilot-mcp/tools/device/physical-device-console/).
</Aside>

<Aside type="tip">
  If a device shows up but its state is not "Available," it may need to be unlocked, trusted, or updated. Check that the device is unlocked and that you have tapped "Trust" on the device when prompted.
</Aside>

<Aside type="note">
  Wi-Fi-paired devices must be on the same network as your Mac and must have been previously paired via USB with the "Connect via network" checkbox enabled in Xcode's Devices & Simulators window.
</Aside>
