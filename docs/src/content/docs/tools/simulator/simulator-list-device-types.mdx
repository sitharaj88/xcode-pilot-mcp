List all device types available for creating simulators, including iPhones, iPads, Apple Watches, Apple TVs, and Apple Vision Pro. Each entry shows the device name, identifier, and supported product family. Use this tool to find the correct `deviceTypeId` before creating a new simulator.

## Parameters

This tool takes no parameters.

## Tips

<Aside type="tip">
  Copy the `Identifier` value (e.g., `com.apple.CoreSimulator.SimDeviceType.iPhone-16-Pro`) directly into the `deviceTypeId` parameter of [`simulator_create`](/xcode-pilot-mcp/tools/simulator/simulator-create/).
</Aside>

<Aside type="tip">
  When testing responsive layouts, create simulators for the smallest and largest devices in your supported range. For example, pair an iPhone SE (3rd generation) with an iPhone 16 Pro Max to cover both extremes of screen size.
</Aside>

<Aside type="note">
  Not all device type and runtime combinations are valid. For example, you cannot create an iPhone 16 simulator with an iOS 16.4 runtime because that device was not available in that iOS version. The `simulator_create` tool will report an error if the combination is invalid.
</Aside>
