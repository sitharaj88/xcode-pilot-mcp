List all iOS simulators with their current state (Booted, Shutdown, Creating, etc.). Use this tool to discover available simulators, check which ones are running, or find a specific device UDID before performing other simulator operations.

## Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `state` | string | No | — | Filter simulators by state. Accepted values: `"Booted"`, `"Shutdown"`, `"Creating"`, `"Shutting Down"`. |

## Usage Examples

### List all simulators

> "Show me all my iOS simulators"

```json
{}
```

### List only booted simulators

> "Which simulators are currently running?"

```json
{
  "state": "Booted"
}
```

### List shutdown simulators

> "Which simulators are available but not running?"

```json
{
  "state": "Shutdown"
}
```

## Tips

<Aside type="tip">
  Use `state: "Booted"` to quickly find which simulators are currently running before launching an app or capturing a screenshot.
</Aside>

<Aside type="note">
  The UDID shown in parentheses is the unique identifier you need for most other simulator tools like `simulator_boot`, `simulator_erase`, and `app_install`.
</Aside>

<Aside type="tip">
  Run `simulator_list` without any parameters first to get an overview of all available simulators, then use the UDID from the output in subsequent commands.
</Aside>
