List all simulator runtimes installed on your system, including iOS, watchOS, tvOS, and visionOS. Each runtime entry shows its name, version, build number, identifier, and availability status. Use this tool to discover which runtimes are available before creating a new simulator.

## Parameters

This tool takes no parameters.

## Tips

<Aside type="tip">
  Copy the `Identifier` value (e.g., `com.apple.CoreSimulator.SimRuntime.iOS-18-0`) directly into the `runtimeId` parameter of [`simulator_create`](/xcode-pilot-mcp/tools/simulator/simulator-create/).
</Aside>

<Aside type="note">
  Runtimes marked as "Not Available" need to be downloaded first. Open Xcode &gt; Settings &gt; Platforms to download additional simulator runtimes.
</Aside>

<Aside type="tip">
  After upgrading Xcode, run this tool to verify that the runtimes you need are still available. Major Xcode updates sometimes drop support for older runtimes.
</Aside>
